# What software is used?

Hey there 👋

We’ll be using two tools:

- **Visual Studio Code**
- **RunJS**

You can download both from their official websites if you'd like. And if you prefer using a different code editor, that’s completely up to you.

---

## Visual Studio Code

**Visual Studio Code** is one of the most popular editors used in professional software development. Later in the course, we’ll use it to build real-world projects and applications.

That said, if you’re just getting started, VS Code can feel a little overwhelming at first — which is why we’ll begin with something much simpler.

---

## RunJS

**RunJS** is a lightweight, beginner-friendly tool designed specifically for practicing JavaScript.

It allows you to:

- Write JavaScript code on the left side
- Instantly see the output on the right side

No setup, no configuration, and no distractions — just pure JavaScript practice.

We’ll mainly use RunJS in the beginning of the course to quickly demonstrate small code examples without needing to create a full project in VS Code.

---

## Do You Need the Paid Version?

RunJS does offer paid features, but I would **not** recommend signing up for them.

The free version is more than enough for everything we’ll do during the beginner phase of the course.

Just keep in mind:

> RunJS is not designed for building real-world applications.
> It’s best used for learning, experimenting, and testing small pieces of JavaScript code quickly.
